# Optimizers

All standard Keras optimizers are supported, but importing custom TensorFlow optimizers won't work:

*  SGD
*  RMSprop
*  Adagrad
*  Adadelta
*  Adam
*  Adamax
*  Nadam
*  TFOptimizer
